  <action>
    The root cause is the salary reserve formula on line 225:
    ```
    p.salaryCap <= team.salaryRemaining - (TOTAL_PICKS - team.picks.length - 1) * 644
    ```
    This reserves 644 (minimum salary) per remaining pick, but when cheap politicians are already drafted by other teams, no one affordable remains.

    Fix with TWO changes:

    1. **Lower the reserve floor for bench rounds (rounds 9-12).**
    Change the reserve calculation so that for bench rounds (when `team.picks.length >= ACTIVE_SLOTS`), use a much lower reserve per remaining pick — e.g., 200 instead of 644. This allows bench picks to use more of the remaining salary:

    ```typescript
    const picksRemaining = TOTAL_PICKS - team.picks.length - 1
    const reservePerPick = team.picks.length >= ACTIVE_SLOTS ? 200 : 644
    const salaryReserve = picksRemaining * reservePerPick
    ```

    Then update the filter condition on line 225 to use `salaryReserve` instead of the inline calculation.

    2. **Make the fallback pick unconditional for the last bench slot.**
    When it is the LAST pick (round 11, index 0-based), remove the salary cap check entirely in the fallback — just take the cheapest available politician regardless. A team should never have fewer than 12 picks. Update lines 232-239:

    ```typescript
    } else {
      // Fallback: take the cheapest available
      const cheapest = available[available.length - 1]
      const isLastPick = team.picks.length === TOTAL_PICKS - 1
      if (cheapest && (isLastPick || cheapest.salaryCap <= team.salaryRemaining)) {
        team.picks.push(cheapest)
        team.salaryRemaining -= cheapest.salaryCap
        available.splice(available.indexOf(cheapest), 1)
      }
    }
    ```

    After making changes, add a validation log at line ~243 (after the draft loop ends) to warn if any team has fewer than TOTAL_PICKS:
    ```typescript
    for (const team of teams) {
      if (team.picks.length < TOTAL_PICKS) {
        console.error(`FATAL: ${team.name} only has ${team.picks.length}/${TOTAL_PICKS} picks`)
        process.exit(1)
      }
    }
    ```
