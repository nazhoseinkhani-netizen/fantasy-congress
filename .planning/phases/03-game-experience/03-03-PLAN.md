'use client' component. Props:
```typescript
interface ActivityFeedProps {
  league: League

  politicians: Politician[]
  trades: Trade[]
}
```

Derive activity events at render time (D-13):

```typescript
type ActivityEvent =
  | { type: 'matchup_result'; week: number; winnerName: string; loserName: string; winnerScore: number; loserScore: number; sortKey: number }
  | { type: 'mvp'; week: number; teamName: string; politicianName: string; points: number; sortKey: number }
  | { type: 'big_trade'; politicianName: string; ticker: string; points: number; date: string; sortKey: number }
```

Generation logic:
1. For each completed matchup in schedules: determine winner/loser by homeScore vs awayScore, add matchup_result event. `sortKey = week * 1000 + 100`.
2. For each weekResult: find MVP (mvpBioguideId) and look up politician name. Add mvp event. `sortKey = week * 1000 + 50`.
3. For trades by politicians on ANY roster in this league: if `trade.fantasyPoints > 20`, add big_trade event. Get all bioguideIds from all team rosters in the league. Filter trades by those IDs. `sortKey = parseInt(trade.disclosureDate.replace(/-/g, ''))`.

Sort events by sortKey descending (most recent first). Show up to 20 events.

Render each event as a compact card:
- matchup_result: Trophy icon + "Week {N}: {winner} beat {loser} {score}-{score}" with winner in `font-semibold`
- mvp: Star icon + "Week {N} MVP: {politicianName} ({points} pts) on {teamName}" with politician name in `text-primary`
- big_trade: Activity icon + "{politicianName} traded {ticker} for {points} pts" with date

Use `lucide-react` icons: Trophy, Star, Activity.

Export: `export function ActivityFeed({ ... })`

Data loading:
```typescript
const [demoState, setDemoState] = useState<DemoState | null>(null)
const [politicians, setPoliticians] = useState<Politician[]>([])
const [trades, setTrades] = useState<Trade[]>([])
const [schedules, setSchedules] = useState<WeekSchedule[]>([])

useEffect(() => {
  Promise.all([
    loadDemoState(),
    loadPoliticians(),
    loadTrades(),
    loadMatchups(),
  ]).then(([demo, pols, trd, { schedules: sched }]) => {
    setDemoState(demo)
    setPoliticians(pols)
    setTrades(trd)
    setSchedules(sched)
  })
}, [])
```

If `!demoState`, show skeleton: 1 large skeleton for standings, 1 for schedule grid, 1 for activity feed.

Derived data:

- `leagueWeekResults = demoState.weekResults.filter(wr => userLeague.teams.some(t => t.id === wr.teamId))`
