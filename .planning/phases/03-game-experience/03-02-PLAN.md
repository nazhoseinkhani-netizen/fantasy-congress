**2. Create `src/components/dashboard/dashboard-page.tsx`:**

'use client' component. No props — loads all data internally.

Data loading in useEffect:
```typescript
const [demoState, setDemoState] = useState<DemoState | null>(null)
const [politicians, setPoliticians] = useState<Politician[]>([])
const [trades, setTrades] = useState<Trade[]>([])
const [schedules, setSchedules] = useState<WeekSchedule[]>([])
const selectedWeek = useGameStore((s) => s.selectedWeek)
const setActiveLeagueId = useGameStore((s) => s.setActiveLeagueId)

useEffect(() => {
  Promise.all([
    loadDemoState(),
    loadPoliticians(),
    loadTrades(),
    loadMatchups(),
  ]).then(([demo, pols, trd, { schedules: sched }]) => {
    setDemoState(demo)
    setPoliticians(pols)
    setTrades(trd)
    setSchedules(sched)
    setActiveLeagueId(demo.activeLeagueId)
  })
}, [setActiveLeagueId])
```

If `!demoState`, show skeleton loading (4 skeleton rectangles for KPIs, 1 large skeleton for matchup area).

Once loaded, derive:
- `userLeague = demoState.leagues.find(l => l.id === demoState.activeLeagueId)`
- `userTeam = userLeague.teams.find(t => t.id === demoState.userTeamId)`
- `leagueSchedules = schedules.filter(s => s.matchups.some(m => m.leagueId === userLeague.id))` (or filter matchups by leagueId)
- `currentWeekSchedule = leagueSchedules.find(s => s.week === selectedWeek)`
- `userMatchup = currentWeekSchedule?.matchups.find(m => m.homeTeamId === userTeam.id || m.awayTeamId === userTeam.id)`
- `opponentTeamId = userMatchup ? (userMatchup.homeTeamId === userTeam.id ? userMatchup.awayTeamId : userMatchup.homeTeamId) : undefined`
- `opponentTeam = userLeague.teams.find(t => t.id === opponentTeamId)`
- `userWeekResult = demoState.weekResults.find(wr => wr.teamId === userTeam.id && wr.week === selectedWeek)`
- `opponentWeekResult = opponentTeamId ? demoState.weekResults.find(wr => wr.teamId === opponentTeamId && wr.week === selectedWeek) : undefined`
- `leagueRank = userLeague.teams.sort((a,b) => b.pointsFor - a.pointsFor).findIndex(t => t.id === userTeam.id) + 1` (sort by pointsFor for rank)
- `nextWeekSchedule = leagueSchedules.find(s => s.week === selectedWeek + 1)`
- `nextOpponentId` derived from nextWeekSchedule, `nextOpponent` team name
- `politicianMap = new Map(politicians.map(p => [p.bioguideId, p]))`
