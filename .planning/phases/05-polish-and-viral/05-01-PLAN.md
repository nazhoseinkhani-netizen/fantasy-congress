- Fill circle: `<motion.circle>` with `stroke={color}`, `strokeDashoffset={dashOffset}`, same dimensions
- `[showScore, setShowScore] = useState(!animateOnMount)` — score number fades in after animation
- `useEffect` with `progress.on('change', (v) => { if (v >= score * 0.95) setShowScore(true) })` to trigger score label appearance
- When `showLabel`, render `<text x="50" y="52" textAnchor="middle" fill="currentColor" className="text-xs font-bold">{Math.round(progress.get())}</text>` and `<text x="50" y="58" textAnchor="middle" fill="currentColor" className="text-[8px] text-muted-foreground">{labelText || tierName}</text>`
