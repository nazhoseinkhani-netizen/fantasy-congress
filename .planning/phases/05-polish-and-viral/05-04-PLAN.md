**Create src/components/share/use-share-card.ts:**
- `'use client'` — no, this is a hook file, not a component. But it uses browser APIs so must be imported by client components.
- Import `{ toPng }` from `'html-to-image'`
- Import `{ useRef, useState, useCallback }` from `'react'`
- Export `useShareCard` hook returning `{ cardRef: RefObject<HTMLDivElement>, generate: () => Promise<string | null>, generating: boolean }`
- `cardRef = useRef<HTMLDivElement>(null)`
- `[generating, setGenerating] = useState(false)`
- `generate` callback:
  - Guard: `if (!cardRef.current) return null`
  - `setGenerating(true)`
  - `try/finally` block with `setGenerating(false)` in finally
  - Options: `{ pixelRatio: 2, cacheBust: true, quality: 0.95 }`
  - iOS Safari workaround: call `toPng(cardRef.current, opts)` three times, discard first two, return third
  - Return the data URL string
- Export helper `downloadImage(dataUrl: string, filename: string)`: creates `<a>` element with `href=dataUrl`, `download=filename`, clicks it, removes it
- Export helper `shareImage(dataUrl: string, title: string)`: if `navigator.canShare` is available, convert dataUrl to Blob via `fetch(dataUrl).then(r => r.blob())`, create `File` with name `'share.png'`, call `navigator.share({ title, files: [file] })`. If not supported, fall back to `downloadImage`.
